# Rohit – Portfolio (Next.js + Tailwind)

## Dev
```bash
npm install
npm run dev
```

## Build

```bash
npm run build && npm start
```

## Edit Content

* `src/data/profile.ts` – name, bio, socials, resume URL.
* `src/data/projects.ts` – project cards.

## Deploy

* Vercel recommended. Add your custom domain. Done.
